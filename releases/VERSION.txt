Quick Start (Windows):
  1. Extract ZIP to C:\Projects\HRPerformance
  2. Run: .\scripts\restore-packages.ps1
  3. Run database scripts 01-10 in SQL Server (HRPerformanceDB)
  4. Set HrIntegration:Password in appsettings.json or user-secrets
  5. .\start.ps1
  6. Open http://localhost:5000
