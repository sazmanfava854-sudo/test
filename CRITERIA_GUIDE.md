# راهنمای معیارهای تصمیم‌گیری

این سند چارچوب معیارهای پژوهش را برای **K-Means → AHP → TOPSIS** تعریف می‌کند.

## دو پروفایل

| پروفایل | تعداد | کاربرد |
|---------|-------|--------|
| `core` | ۸ معیار | مطابق متن پروپوزال |
| `extended` | ۱۴ معیار | توسعه‌یافته — برای تحلیل عمیق‌تر و AHP چندلایه |

```bash
python3 scripts/clean_and_normalize.py --profile core
python3 scripts/clean_and_normalize.py --profile extended
python3 scripts/clean_and_normalize.py --profile all      # پیش‌فرض
```

## گروه‌بندی برای AHP (سلسله‌مراتب معیارها)

در `data/criteria_config.json` معیارها به ۶ گروه بالادستی نگاشت شده‌اند:

```
اقتصادی          → CAPEX, OPEX
انرژی            → مصرف انرژی
کیفیت ارتباط     → نرخ داده، تأخیر، دوبلکس، پهنای باند
پوشش و مقیاس    → برد، لینک بودجه، اندازه شبکه
زیرساخت و طیف   → سلولی، فرکانس، کارایی طیفی
امنیت            → امتیاز امنیت
```

این ساختار با پرسشنامه پروپوزال (پوشش موبایل، بودجه، فاصله سنسورها، …) هم‌راستا است.

## ستون‌های متادیتا (خارج از مدل)

این فیلدها در CSV خام باقی می‌مانند ولی وارد K-Means نمی‌شوند:

- کاربرد اصلی
- شرایط تست
- مدولاسیون (خام)
- نوع طیف (خام)

## تنظیم معیارهای دلخواه

فایل `data/criteria_config.json` را ویرایش کنید:

```json
"extended": {
  "criteria": [
    "capex_usd",
    "energy_mw",
    ...
  ]
}
```

معیار را اضافه/حذف کنید، سپس اسکریپت را دوباره اجرا کنید.

---

## پیشنهاد برای پروپوزال

- **فصل روش تحقیق:** پروفایل `core` برای K-Means (مطابق متن فعلی)
- **فصل توسعه / بحث:** پروفایل `extended` برای نشان دادن عمق تحلیل
- **AHP:** از گروه‌های ۶گانه به‌عنوان سطح دوم سلسله‌مراتب استفاده کنید
